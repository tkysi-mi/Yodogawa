## チェックリスト

## レビュー確認質問

- 「各技術の選定理由は要件に基づいていますか？」
- 「チームの習熟度に対して無理のない選定ですか？」
- 「バージョンは LTS や安定版が選ばれていますか？」
- 「監視・テスト戦略も含まれていますか？」

## エスカレーション時の推奨応答

- **ユーザーが決められない**: 「最も標準的でリスクの少ない [技術名] を仮採用とし、実装フェーズで再評価しませんか？」
- **コスト・学習コストの懸念**: 「初期フェーズは慣れた技術（[技術名]）を採用し、複雑要件が出てから移行検討しましょう。」
