# コミット・PR テンプレート

SKILL.md 手順2.2（ステップコミット）・3.3（フェーズコミット）・6（最終コミットとPR）で使うテンプレート。

## ステップコミット

```
feat(task-{id}): {ステップ名}
```

## フェーズ完了コミット

```
feat(task-{id}): complete phase {N} - {フェーズ名}

- {ステップ1の概要}
- {ステップ2の概要}
- {ステップ3の概要}

受け入れ基準:
- [x] {基準1}
- [x] {基準2}
- [x] {基準3}
```

## 最終コミット

```
feat(task-{id}): implement {タスク概要}

実装内容:
- {変更1}
- {変更2}
- {変更3}

受け入れ基準:
- [x] {基準1}
- [x] {基準2}
- [x] {基準3}

テスト:
- ユニットテスト: 100% 通過
- 統合テスト: 100% 通過
- E2E テスト: 100% 通過
- カバレッジ: 85%

関連ドキュメント:
- docs/tasks/task{id}-{スラッグ}/a-definition.md
- docs/tasks/task{id}-{スラッグ}/b-research.md
- docs/tasks/task{id}-{スラッグ}/c-implementation.md
```

## PR タイトル

```
feat(task-{id}): {タスク概要}
```
